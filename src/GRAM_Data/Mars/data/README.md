# MarsGRAM data

Files and Subfolders:
- ASCII: Readable text versions of the Mars data.
- MGCM_lower_data.bin: Lower atmosphere data for the MGCM model.
- MGCM_surface_data.bin: Surface data for the MGCM model.
- MGCM_upper_data.bin: Upper atmosphere data for the MGCM model.
- MOLA_data.bin: Mars Orbiter Laser Altimeter topographical data.
- TES_dust_data.bin: Dust tau data for the TES model.
- TES_lower_data.bin: Lower atmosphere data for the TES model.
- TES_surface_data.bin: Surface data for the TES model.
- TES_upper_data.bin: Upper atmosphere data for the TES model.
