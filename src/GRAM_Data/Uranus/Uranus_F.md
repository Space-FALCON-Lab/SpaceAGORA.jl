The UranusGRAM F Interface
===========================

This documents the FORTRAN interface for UranusGRAM.  Since the GRAM framework is common to all models, only the Uranus specific information is documented on this page.  See the [FORTRAN interface](Documentation/Markdown/GRAM_F.md) page for documentation on the common framework.

A simple example of using the Uranus atmosphere can be seen [here](@ref Uranus/examples/Uranus_F.f90).

- - - - - - - 

There are no Uranus specific settings.  The [common framework](Documentation/Markdown/GRAM_F.md) contains all interface functions.
