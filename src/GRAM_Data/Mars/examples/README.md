# MarsGRAM Examples

Files and Subfolders:
- MarsGRAM.cpp: The MarsGRAM main program.
- ExampleRunner.cpp: Runs a number of examples from common/examples.
- Mars.cpp: A C++ example.
- Mars_C.c: A C example.
- Mars_F.f90: A Fortran example.
- MarsRad.cpp: Program to compute longwave and shortwave fluxes at the surface and top of the atmosphere of Mars.
