The JupiterGRAM C++ Interface
=============================

This documents the \ref GRAM::JupiterAtmosphere interface class for JupiterGRAM.  Since the GRAM framework is common to all models, only the Jupiter specific information is documented on this page.  See the [C++ interface](Documentation/Markdown/GRAM.md) page for documentation on the common framework.

A simple example of using the Jupiter atmosphere class can be seen [here](@ref Jupiter/examples/Jupiter.cpp).

- - - - - - - 

There are no Jupiter specific settings.  The [common framework](Documentation/Markdown/GRAM.md) contains all interface functions.
