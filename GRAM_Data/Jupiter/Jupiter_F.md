The JupiterGRAM F Interface
===========================

This documents the FORTRAN interface for JupiterGRAM.  Since the GRAM framework is common to all models, only the Jupiter specific information is documented on this page.  See the [FORTRAN interface](Documentation/Markdown/GRAM_F.md) page for documentation on the common framework.

A simple example of using the Jupiter atmosphere can be seen [here](@ref Jupiter/examples/Jupiter_F.f90).

- - - - - - - 

There are no Jupiter specific settings.  The [common framework](Documentation/Markdown/GRAM_F.md) contains all interface functions.
