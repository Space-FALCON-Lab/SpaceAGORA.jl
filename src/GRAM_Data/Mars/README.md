# MarsGRAM

To build MarsGRAM using a Linux/MinGW makefile, navigate to the folder /GRAM/Builds and follow the README instructions.

To build MarsGRAM using MSVC++ 2017 or 2015, follow the instructions in the README file in \GRAM\MSVS.

Files and Subfolders:
- data: Binary data files for the Mars models.
- examples: Examples and the GRAM program for this model.
- include: Header files for the model.
- sample_inputs: Sample input and output files.
- source: Souce code for the model.
- unittest: Source code for unit tests.
- md files: Markdown files used to build the Programmer's Guide.
