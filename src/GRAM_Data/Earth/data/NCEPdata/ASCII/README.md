# EarthGRAM NCEP data in ASCII format

Files and Subfolders:
- NCEPbin.bat: Windows command file to build NCEPbin.cpp using g++.
- NCEPbin.cpp: C++ utility to convert ASCII NCEP files.
- NCEPbinF.bat: Windows command file to build NCEPbinF.f90 using gfortran.
- NCEPbinF.f90: Fortran utility to convert ASCII NCEP files.
- NCEPverify.bat: Windows command file to build NCEPverify.cpp using g++.
- NCEPverify.cpp: C++ utility to verify equality of binary files created by NCEPbin and NCEPbinF.
- Nfyyyymm.txt: NCEP data file for POR yyyy and month mm.
