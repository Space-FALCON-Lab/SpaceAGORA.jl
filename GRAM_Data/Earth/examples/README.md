# EarthGRAM Examples

Files and Subfolders:
- EarthGRAM.cpp: The EarthGRAM main program.
- EarthCorrMonte.cpp: The EarthGRAM CorrMonte main program.
- EarthCorrTraj.cpp: The EarthGRAM CorrTraj main program.
- EarthCorrMulti.cpp: An example of correlating multple trajectories.
- ExampleRunner.cpp: Runs a number of examples from common/examples.
- Earth.cpp: A C++ example.
- Earth_C.c: A C example.
- Earth_F.f90: A Fortran example.
