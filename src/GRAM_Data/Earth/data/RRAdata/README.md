# EarthGRAM RRA data

Files and Subfolders:
- rrasites.txt: ASCII database of RRA data by site and year contained in this folder.
- T1sssyy.txt: RRA winds data for site sss and year yy.
- T2sssyy.txt: RRA dynamics data for site sss and year yy.
- T3sssyy.txt: RRA psychrometric data for site sss and year yy.
