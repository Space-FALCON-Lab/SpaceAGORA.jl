Grid specifications for 2.0-degree MERRA-2 data files.
Grid specification required to read files correctly.

Pressures: Number of pressure levels
Latitudes: Number of latitude data points
Longitudes: Number of longitude data points
Surface Longitudes: Number of longitude data points for surface data
